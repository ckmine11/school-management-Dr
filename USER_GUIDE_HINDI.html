अध्याय 1

सिस्टम का परिचय

यह एक स्कूल मैनेजमेंट सिस्टम है जिससे स्कूल के सभी काम आसानी से ऑनलाइन किए जा सकते हैं — कहीं से भी, किसी भी device से।

इस सिस्टम में क्या-क्या होता है?

कामविवरण
छात्र प्रबंधननए छात्र जोड़ना, जानकारी देखना, ID Card बनाना
शिक्षक प्रबंधनशिक्षकों का रिकॉर्ड रखना
उपस्थितिरोज़ की हाजिरी लगाना
फीसफीस जमा करना, PDF रसीद देना
परिणामपरीक्षा के नंबर डालना, grade automatic
समय-सारणीक्लास का टाइमटेबल बनाना
परीक्षा कार्यक्रमपरीक्षा की तारीख, समय, reminder
WhatsAppParents को automatic messages भेजना
सूचनाएंस्कूल की Notice सभी तक पहुंचाना

Teacherशिक्षकअपनी class की attendance और results
Studentछात्रसिर्फ अपनी जानकारी देख सकते हैं
Parentअभिभावकअपने बच्चे की जानकारी देख सकते हैं
अध्याय 2

Login कैसे करें

Step 1 — Browser में खोलें

अपने phone या computer के browser (Chrome, Firefox) में यह address खोलें:

http://आपके-सर्वर-का-IP/login.html उदाहरण: http://152.67.10.25/login.html

Step 2 — Email और Password डालें

1
Email box में अपना email address डालें
2
Password box में अपना password डालें
3
Sign In button दबाएं

Step 3 — Dashboard खुलेगा

Login होने के बाद आप अपने role के अनुसार Dashboard पर पहुंच जाएंगे।

✅ पहली बार Login के बाद: अगर "Please change your password" दिखे तो नया password (कम से कम 8 अक्षर) डालें और Save करें।

Default Admin Login (पहली बार)

3.10 WhatsApp Setup

2
10-15 seconds रुकें — QR Code दिखेगा

अध्याय 9

🔧 सामान्य समस्याएं और समाधान

❌ Login नहीं हो रहा

समस्यासमाधान
Email गलत हैAdmin से सही email पूछें
Password गलत हैAdmin से password reset करवाएं
Account inactive हैAdmin से account activate करवाएं
Browser cache issueIncognito / Private mode में try करें

❌ WhatsApp Messages नहीं आ रहे

1. Student record में Parent Phone सही डला हो — सिर्फ 10 अंक (जैसे 9876543210)
2. Admin → WhatsApp Setup → Status "Connected ✓" होना चाहिए
3. देश का code (91) नहीं डालना — system खुद लगाता है

❌ Fee Receipt / ID Card Download नहीं हो रही

Browser में Popup Blocker बंद करें → Settings → Allow popups for this site

❌ Timetable नहीं दिख रहा

Admin ने उस class का timetable बनाया होगा तभी दिखेगा।
Admin → Timetable → Class select करें → timetable बनाएं।

❌ Page नहीं खुल रहा

1. Internet connection check करें
2. Server IP सही है या नहीं देखें
3. Browser cache clear करें (Ctrl + Shift + Delete)

❌ Password भूल गए

Student / Teacher / Parent → Admin से reset करवाएं
Admin → Login page पर Recovery Secret use करें
